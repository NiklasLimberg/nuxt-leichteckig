# leichteckig

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Run tests

```
# Run all tests
$ npm run test

# Start test watcher for development
$ npm run test-watch
```
